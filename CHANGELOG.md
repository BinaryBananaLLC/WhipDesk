# Changelog

Release notes are generated automatically for every tagged release — the full history lives on
the [GitHub Releases page](https://github.com/BinaryBananaLLC/WhipDesk/releases).

Each release ships signed multi-platform binaries (macOS notarized `.pkg`, Windows
Authenticode `Setup.exe`, Linux `.tar.gz`), `SHA256SUMS`, and SLSA build provenance — see
[docs/VERIFYING-DOWNLOADS.md](docs/VERIFYING-DOWNLOADS.md).
